# UNIX VirtualBox-Kit
UNIX VirtualBox Kits to quickly deploy a small x86_64 virtual machine.
### Machines available

1. Arch
2. Artix OpenRC (coming soon)
3. Debian
4. Devuan OpenRC (coming soon)
5. Fedora (coming soon)
6. Rocky (coming soon)
7. Alpine (coming soon)
8. Void (coming soon)
9. Chimera (coming soon)
10. FreeBSD
11. OpenBSD
12. NetBSD (coming soon)

### Virtualbox settings

* Users [username : password]
  - root : root
  - user : user
* hostname: unix-workstation
* 64GiB hard drive, GPT, EFI
* VirtualBox guest additions installed (if available)
